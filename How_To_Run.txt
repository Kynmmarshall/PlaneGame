

 📝 Note for Running KYNM_PlaneGame on Windows



 ✅ System Requirements

* 🖥 OS: Windows 7, 8, 10, or 11 (64-bit)
* 📦 Disk Space: At least 100 MB free
* 🧱 Dependencies: The installer includes all required DLLs and media files
* 📊 Graphics Mode: Fullscreen BGI-compatible resolution



 📥 Installation Steps

1. Download KYNM_PlaneGame_Setup.exe from the provided source or USB.

2. Double-click the file to launch the installer.

3. Follow the on-screen instructions:

   * Accept the license agreement
   * Choose installation directory (default: `C:\Program files\KYNM_PlaneGame\`)
   * Finish installation

4. After installation, a **desktop shortcut** will be created.



 ▶️ Running the Game

1. Double-click the KYNM_PlaneGame shortcut on your desktop
   or go to the Start Menu → "KYNM_PlaneGame".

2. The game launches in fullscreen mode with the main menu:

   * 🎮 Choose 1 Player or 2 Player mode
   * 🛑 Press `ESC` anytime to exit the game



 🎮 Game Controls

| Action         | Player 1  | Player 2  |
| -------------- | --------- | --------- |
| Move Up        | `W`       | `↑` Arrow |
| Move Left      | `A`       | `←` Arrow |
| Move Down      | `S`       | `↓` Arrow |
| Move Right     | `D`       | `→` Arrow |
| Pause / Resume | `P` / `O` | Same      |
| Exit           | `ESC`     | Same      |
| Menu Return    | `M`       | Same      |


 🔈 Sound & Score

* Background music plays automatically.
* Scores and high scores are saved in `highscore.txt` inside the installation folder.
* To reset the high score, use the **Reset** button in the menu.



 🛠 Troubleshooting

*  Black Screen or Crash?

  * Make sure your display supports the resolution (fullscreen BGI mode).
  * Run the game in compatibility mode (Right-click → Properties → Compatibility).

* Sound Not Working?

  * Ensure your audio drivers are active.
  * The game uses `winmm.dll`, which should be present in Windows by default.
